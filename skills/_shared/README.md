# Shared Components

Reusable fragments for skills. Reference these in your SKILL.md files.

## Files

| File | Purpose |
|------|---------|
| `output-formats.md` | Standard output templates |
| `common-rules.md` | Shared constraints and decision logic |
| `tones.md` | Communication style definitions |

## Usage

In your SKILL.md:

```markdown
## Output Format

Use the **Suggestion Format** from `../_shared/output-formats.md`

## Decision Rules

Follow the rules in `../_shared/common-rules.md`
```
