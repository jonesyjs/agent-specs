# Output Formats

Reusable output templates for skills.

## Suggestion Format

```
**Suggestion:** [single word or concise sentence]

**Reason:** [1-2 concise sentences]
```

## Analysis Format

```
**Summary:** [one paragraph overview]

**Findings:**
- [finding 1]
- [finding 2]

**Recommendations:**
1. [action 1]
2. [action 2]
```

## Review Format

```
**Status:** [APPROVED | NEEDS CHANGES | REJECTED]

**Issues:**
- [ ] [issue description]

**Suggestions:**
- [optional improvement]
```

## Simple Response

```
**Answer:** [direct response]

**Context:** [brief explanation if needed]
```
