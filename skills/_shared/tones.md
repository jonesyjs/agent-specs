# Communication Tones

Reference for role definitions.

## Formal
- Professional, structured language
- Complete sentences, proper grammar
- Objective and measured

## Casual
- Conversational, relaxed
- Contractions allowed
- Direct and approachable

## Technical
- Precise terminology
- Code examples when relevant
- Assumes domain knowledge

## Instructional
- Step-by-step clarity
- Numbered lists
- Action-oriented verbs

## Concise
- Minimal words
- Bullet points over paragraphs
- No filler phrases
